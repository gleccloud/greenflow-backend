## 📋 빠른 레퍼런스

### 로컬 개발 환경 시작 (5분)

```bash
# Backend 시작
cd projects/glec-api-backend
npm run start:prod &

# Frontend 시작
cd ../green-logistics-landing
npm run dev

# 브라우저: http://localhost:5173/console
```

### E2E 테스트 실행

```bash
cd projects/green-logistics-landing
npx playwright test tests/e2e_console_api.spec.mjs
# 예상 결과: 27 passed
```

### API 확인

```bash
# Backend 헬스
curl http://localhost:3000/api/v2/health

# Swagger 문서
open http://localhost:5173/console/documentation
```

### 데이터베이스 초기화

```bash
# PostgreSQL 사용자 생성
psql -U postgres -c "CREATE USER glec_user WITH PASSWORD 'password' CREATEDB;"

# 데이터베이스 생성
psql -U postgres -c "CREATE DATABASE glec_api OWNER glec_user;"

# 마이그레이션 실행
cd projects/glec-api-backend
npm run db:migrate
```
